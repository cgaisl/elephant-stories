Connect a story repo

This reader renders graded-reader stories straight from a GitHub repo — one with a stories/ folder and a VOCABULARY.md (create yours from the template in this repo).

For a private repo: GitHub → Settings → Developer settings → Fine-grained personal access tokens → new token, scoped to only that repo with Contents: Read-only. Both values are stored only in this browser (localStorage) and sent only to api.github.com.
